# 2026 世界新台商：出海實戰導航守則
### 眾達國際法律事務所 (Jones Day) × 商業周刊 聯合製作

## 專案概述

本專案為眾達國際法律事務所與商業周刊合作的 **2026 年度企劃** 落地頁（Landing Page），以「出海實戰導航守則」為主題，提供台商出海所需的全球法律策略資訊，協助企業在全球化布局中降低法律風險。

### 色彩變數
```css
--red: #C8102E;          /* 主品牌紅 */
--red-dark: #9B0D22;     /* 深紅（hover 用）*/
--red-light: #F5C6CE;

--gold: #D4A017;
--off-white: #F7F5F2;
--pink-bg: #FDE8EB;      /* icon 背景 */
```

### 動畫效果
| 名稱 | 用途 |
|------|------|
| `fadeInUp` | Hero 區塊各元素入場（stagger 0.1s 延遲）|
| `blink` | Hero badge 左側指示燈 |
| `pulse-ring` | Globe rings 脈衝縮放 |
| `rotate-slow` | Why 區塊外圓慢速旋轉 |
| `.reveal` / `IntersectionObserver` | 頁面滾動進場動畫 |
